## Cloud

1. Fork the repo.
2. Create a new project in Cloud.
3. Connect to your warehouse.
4. Connect to your fork of the repo.
5. Run dbt seed.
6. Run dbt deps.
7. Delete the sample-data.
8. Run dbt build.
